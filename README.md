# Formula 1 Race Analytics Dashboard

### Australian Grand Prix 2026 -- Data Analysis Project

This project analyzes the **2026 Australian Grand Prix (Albert Park
Circuit)** using a data-driven dashboard.\
The goal is to extract insights about **driver performance, race
strategy, and team pace** using lap-by-lap race data.

The project combines **Python-based data extraction** with **Tableau
visualization** to create an interactive race analytics dashboard.

------------------------------------------------------------------------

# 📊 Dashboard Overview

The dashboard explores multiple aspects of the race including:

-   Race results and key performance indicators
-   Fastest lap and sector analysis
-   Constructor pace comparison
-   Driver lap performance heatmap
-   Position changes during the race
-   Tyre strategy visualization

These visualizations help explain **how strategy, pace, and race
dynamics influenced the final result**.

------------------------------------------------------------------------

# 🔎 Key Insights from the Race

### Mercedes Dominance

Mercedes secured a **1--2 finish** with George Russell winning the race
and Kimi Antonelli finishing second.

### Verstappen's Exceptional Drive

Max Verstappen recorded the **fastest lap of the race (1:22.091)** and
gained several positions compared to his starting grid position, showing
strong race pace and overtaking ability.

### Strategy Played a Major Role

Most drivers followed a **Medium → Hard one-stop tyre strategy**,
highlighting strong tyre durability at Albert Park.

### Race Dynamics

Position change analysis shows that **most overtakes occurred in the
early stages of the race**, after which positions stabilized.

### Sector Performance Differences

Sector analysis reveals how different drivers gained time in specific
parts of the track, reflecting car setup and track characteristics.

# 📈 Dashboard Features

### Race Summary

Displays the winner, fastest lap, and podium finishers.

### Constructor Pace Analysis

Compares average lap times across teams to evaluate race pace.

### Driver Performance Heatmap

Visualizes lap time variations across the race.

### Position Change Visualization

Tracks driver positions throughout the race to highlight overtakes.

### Tyre Strategy Analysis

Shows tyre compound usage and pit stop strategies.

### Sector Performance

Compares the fastest sector times across drivers.
